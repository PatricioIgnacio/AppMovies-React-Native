### Descarga

1. Clonar el repositorio:
   ```sh
   git clone https://github.com/PatricioIgnacio/AppMovies-React-Native
   ```
2. Instalar paquetes NPM:
   ```sh
   npm install
   ```
3. Agregar el llamado a la API en "contents/api":
   ```js
   const API_KEY = "Agregar API";
   ```
